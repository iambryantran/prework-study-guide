HTML

  • HTML is used to structure a language. It is made of elements that work together to provide content and a framework for all pages on the web.
  • Element: A type of content on a page.
  • Elements also divide the content on the page and can describe the content to browsers and other developers.
  • HTML Elements usually contain tags, opening and closing brackets, “<>”. Within the tags can be content or even other nested elements.
  • DOCTYPE html tells the browser to expect an HTML document.
  • html lang="en" this specifies English. The lang= part is an attribute.
  • meta charset="UTF-8" sets the character set to UTF-8.
  • ul is an unordered list that uses bullet points.
  • ol is an ordered list that will use numbers.
  • li is a list item.
  • Semantic elements are named after what they do.
  • link rel="stylesheet" href="./assets/style.css" is used in the head to link to our CSS stylesheet.
  • script src="./assets/script.js" is used at the end of the body to link to our JS script.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is a programming language used by developers to define how webpage content is presented to users. CSS defines the colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • External CSS is typically preferred due to flexibility. It allows you to keep your CSS rules separate from your HTML file.
  • CSS Box Model: In CSS everything is a box.
  • Class: This attribute allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. Then the class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.
  • Selector(s) define the element or attribute in which the rules will apply. These can be things like * (all visible elements), div, elements, or classes.
  • Declaration blocks are the rules that you want to apply to the selector. These are contained within curly brackets.
  • Properties are the attributes of the selector that you want to change. These can be things like color, font-size, or background-color.
  • Values are the values of the properties. These can be things like red, 16px, or #000000.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Block is a value that assigns each image to start on a new line.
  • CSS Structure: selector1, selector2 {property: value;}

Git

  • GitHub allows multiple devs to work on a single project at the same time, reduces the risk of duplicative or conflicting work, and can help decrease production time. Developers can build code, track changes, and innovate solutions to problems that might arise during the site development process simultaneously.
  • Repository (Repo): A folder in which all files and their version histories are stored.
  • Branch: A copy of your repo.
  • Main/Master Branch: This is the "live" version of your code.
  • Development Branch: This is the working version of your code. This branch may consist of multiple feature branches.
  • Feature Branch: This is a working branch for one feature.
  • Issue: Allow users to report new tasks, content fixes, and to track progress on a project board from beginning to end of a specific project.
  • Commit: These are confirmed changes to your code. Commits should be one related change.
  • Pull Request: A request for your pushed commits to be reviewed and merged.
  • Do not forget to "git pull" after a merge to update locally!
  • git clone [URL]: clones a repo from GitHub
  • git remote -v: confirm fetch and push connection to repo
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add file-name: adds file (or -A for all) to your staging area
  • git reset file-name: removes a file from your staging area
  • git commit -m "description": essentially takes a snapshot of your staging area
  • git pull origin branch-name: pulls in the target branch to make sure you are up to date
  • git push origin branch-name: sends my snapshot to the GitHub repo

JavaScript

  • JS is a high-level dynamic programming language that allows users to interact with websites.
  • JavaScript is what turns a static HTML/CSS webpage interactive.
  • Boolean statements are true/false statements. All values are truthy unless they are defined as falsy.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional Statements and Loops: Controls if and when code is executed.
  • = is used to assign a variable.
  • == is used to compare two values.
  • === is used to compare two values and their data types.
  • === is preferred over == so you don't run into accidental conflicting data types.
  • != is used to compare two values and check if they are not equal.
  • !== is used to compare two values and check if they are not equal and not the same data type.
  • var is used to declare a variable. Function-scoped, unless declared outside of function.
  • let is used to declare a variable that can be reassigned. This is block-scoped.
  • const is used to declare a variable that cannot be reassigned. This is block-scoped.
  • var is less common due to bugginess and the advent of let and const.
  • Array: A single variable that holds a group of related data. Arrays are not limited to one type of data. var studentInfo = [Bryan, 28, true];
  • console.log() is a method, and within the parenthesis will be an argument.
  • Function: A set of instructions that tells the computer how to perform a certain task.
  • Functions must be called in order to execute.
  • For Loops can be used in situations where you need to repeatedly execute a block of code a specific number of times or until a condition is met.
